# iPod nano (2nd generation inspired)

This runtime model is a deterministic export of the checked-in
`source/ipod.blend` scene generated by `source/build_ipod.py`. It does not
contain downloaded community geometry or image textures. The modeled labels
and transport symbols use Blender's bundled Bfont, a modified DejaVu Sans
derivative; `BFONT-LICENSE.txt` preserves its Bitstream Vera, Arev, and public
domain notices. No font file is embedded in the runtime GLB. The form-reference
images used while authoring the Blender scene are not included in this Stage
package.

`export_runtime.py` retains the authored body, screen backing, click wheel,
labels, end caps, and Hold switch; excludes the static modeled LCD menu,
studio, lights, and cameras; and adds one full-UV semantic surface for the live
PocketJS framebuffer. `SOURCE.json` records the reviewed source hash and the
export boundary.

The checked-in `ipod-nano-2.glb` contains 22,303 triangles and has SHA-256
`5cfa2a22f0cd8c03f2309109542c407d2d3d05de797cf14e3396b7503287f510`.

The original model geometry and exporter are distributed under the PocketJS
repository's MIT license. Font-derived glyph outlines retain the notices in
`BFONT-LICENSE.txt`.
