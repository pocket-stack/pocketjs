# PocketJS Brand Avatars

White-background 1024x1024 avatar exports for GitHub and X/Twitter profiles.

- `pocketjs-avatar-white-polished.png` is the default recommendation.
- `pocketjs-avatar-white-plate.png` keeps the mark extra safe under circular crops.
- `pocketjs-avatar-white-minimal.png` has the strongest small-size contrast.

The matching `.svg` files are the editable sources for each PNG.
