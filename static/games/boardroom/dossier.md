# BOARDROOM — research dossier

The game adapts the November 2023 OpenAI board crisis. Dates, sequence, and
every quoted line below are from the public record; the rest of the game's
dialogue is original parody. Corrections welcome — the game treats this file
as its source of truth.

## Timeline the game follows

|---|---|---|

  firing — the game does not use them.

  documented Oct 2023 paper dispute.

## Fairness notes

Public figures, newsworthy events, short verbatim quotes with in-game
attribution; everything else is invented comedy, and the game says so on the
title card. No real conversation is reconstructed as if verbatim; the battle
is a menu, not a deposition.
